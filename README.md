## Install
[![NPM](https://nodei.co/npm/polix-cli.png?compact=true)](https://nodei.co/npm/polix-cli/)
> `npm i polix-cli -g`

## Usage

```bash
$ pol -h


  Usage: pol <command>

  Options:

    -v, --version  output the version number
    -h, --help     output usage information

  Commands:

    init           init Polix Dev

...
```

## Getting Started
```bash
$ npm i polix-cli -g
$ pol init example && cd example
$ make build && make dev
```
